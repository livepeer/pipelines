1. Navigate to the `custom_nodes` directory in your ComfyUI workspace
<Tabs>
    <Tab title="RunPod (Linux)">
    ```bash
    cd /workspace/comfyRealtime/ComfyUI/custom_nodes
    ```
    </Tab>

    <Tab title="Local (Windows)">
    ```bash
    cd "$env:HOMEPATH/comfyRealtime/ComfyUI/custom_nodes"
    ```
    </Tab>
</Tabs>

2. Clone the repository
```bash
git clone https://github.com/ad-astra-video/ComfyUI-Florence2-Vision.git
cd ComfyUI-Florence2-Vision
git checkout 0c624e61b6606801751bd41d93a09abe9844bea7
```

3. Install requirements
<Tabs>
    <Tab title="RunPod (Linux)">
    ```bash
    conda activate comfyui
    pip install -r requirements.txt
    ```
    </Tab>

    <Tab title="Local (Windows)">
        ```bash
        pip install -r requirements.txt
        ```
    </Tab>
</Tabs>

4. Download sample workflow for Florence2 [examples/florence2-api.json](https://github.com/ad-astra-video/ComfyUI-Florence2-Vision/blob/main/examples/florence2-api.json) and open it in ComfyUI

5. Proceed to [Install Nodes in ComfyStream](../../reference/custom-node-installation#install-nodes-in-comfystream) each time you add a new node to ComfyUI
