To install a custom node, you will need to install it in both conda/python environments.  

- Since we start ComfyUI in the `comfyui` Conda environment, you can just install nodes via the manager and they will be installed in the correct environment
- You will need to install additional nodes manually to the ComfyStream environment. You can do that by following these steps:

1. Run the following command to initialize Conda (if necessary)

```bash
eval "$(/workspace/miniconda3/bin/conda shell.bash hook)"
```

2. Activate the comfystream environment and install custom nodes:
```bash
conda activate comfystream
cd comfystream
python install.py -- workspace /path/to/comfyui
```

## Compiling TensorRT engines
Some Stable Diffusion models use tensorrt engines that need to be compiled. 

To do this, navigate to the parent directory of the `models` folder used by comfyui and run the following script which will download, build and install the necessary TensorRT engines for our current nodes like `DepthAnything` and `StableDiffusion`

```
cd comfyui
docker pull livepeer/ai-worker:live-app-comfyui
curl -s https://raw.githubusercontent.com/livepeer/ai-worker/main/runner/dl_checkpoints.sh | bash -s -- --tensorrt
```
